Pour que la photo soit ronde et à l'intérieur du cercle que vous avez dessiné, vous pouvez utiliser la technique de découpage de l'image en utilisant un masque circulaire. Voici comment vous pouvez le faire en utilisant le module pdfkit (je suppose que vous utilisez ce module en fonction de votre code) :

javascript
const PDFDocument = require('pdfkit');
const fs = require('fs');

// Créez un nouveau document PDF
const doc = new PDFDocument();
const picturePath = 'chemin/vers/votre/image.jpg';

// Définissez les coordonnées du cercle
const circleX = 125;
const circleY = 130;
const circleRadius = 60;

// Définissez les coordonnées de l'image
const imageX = circleX - circleRadius;
const imageY = circleY - circleRadius;
const imageWidth = circleRadius * 2;
const imageHeight = circleRadius * 2;

// Définissez la couleur de la bordure du cercle
const whiteStrokeColor = "#FFFFFF";
doc.strokeColor(whiteStrokeColor);

// Dessinez le cercle
doc
  .circle(circleX, circleY, circleRadius)
  .lineWidth(2) // Largeur de la bordure
  .stroke();

// Chargez l'image à l'intérieur du cercle
if (fs.existsSync(picturePath)) {
  // Appliquez un masque circulaire à l'image
  doc.save()
     .circle(circleX, circleY, circleRadius) // Créez un masque circulaire
     .clip() // Appliquez le masque
     .image(picturePath, imageX, imageY, { width: imageWidth, height: imageHeight }) // Ajoutez l'image
     .restore(); // Restaurez l'état du document pour supprimer le masque
}

// Enregistrez ou affichez le document PDF comme nécessaire
doc.pipe(fs.createWriteStream('output.pdf')); // Enregistrez le PDF dans un fichier
doc.end(); // Terminez le document

Ce code créera un cercle, puis utilisera le cercle comme masque pour l'image, ce qui la rendra ronde et la positionnera à l'intérieur du cercle. Assurez-vous de remplacer 'chemin/vers/votre/image.jpg' par le chemin réel de votre image. Vous pouvez également ajuster les coordonnées et la taille du cercle selon vos besoins.

Si votre photo est toujours tournée de 90 degrés vers la droite, cela peut être dû à l'orientation EXIF de l'image. Les images peuvent contenir des informations EXIF sur leur orientation, ce qui peut provoquer une rotation incorrecte lors de leur affichage. Pour résoudre ce problème, vous pouvez utiliser une bibliothèque telle que sharp pour détecter et corriger automatiquement l'orientation de l'image avant de l'ajouter au PDF. Voici comment vous pouvez le faire :

Assurez-vous d'installer la bibliothèque sharp en exécutant npm install sharp si vous ne l'avez pas déjà fait.

javascript
const PDFDocument = require('pdfkit');
const fs = require('fs');
const sharp = require('sharp'); // Importez la bibliothèque sharp

// Créez un nouveau document PDF
const doc = new PDFDocument();
const picturePath = 'chemin/vers/votre/image.jpg';

// Définissez les coordonnées du cercle
const circleX = 125;
const circleY = 130;
const circleRadius = 60;

// Définissez le padding pour l'image
const padding = 5;

// Définissez les coordonnées de l'image (en ajoutant le padding)
const imageX = circleX - circleRadius + padding;
const imageY = circleY - circleRadius + padding;
const imageWidth = (circleRadius - padding) * 2;
const imageHeight = (circleRadius - padding) * 2;

// Définissez la couleur de la bordure du cercle
const whiteStrokeColor = "#FFFFFF";
doc.strokeColor(whiteStrokeColor);

// Dessinez le cercle
doc
  .circle(circleX, circleY, circleRadius)
  .lineWidth(2) // Largeur de la bordure
  .stroke();

// Chargez l'image à l'intérieur du cercle après correction de l'orientation
if (fs.existsSync(picturePath)) {
  // Utilisez sharp pour corriger l'orientation de l'image
  sharp(picturePath)
    .rotate() // Corrige l'orientation
    .toBuffer()
    .then((buffer) => {
      // Appliquez un masque circulaire à l'image
      doc.save()
        .circle(circleX, circleY, circleRadius - padding) // Créez un masque circulaire avec le padding
        .clip() // Appliquez le masque
        .image(buffer, imageX, imageY, { width: imageWidth, height: imageHeight }) // Ajoutez l'image
        .restore(); // Restaurez l'état du document pour supprimer le masque

      // Enregistrez ou affichez le document PDF comme nécessaire
      doc.pipe(fs.createWriteStream('output.pdf')); // Enregistrez le PDF dans un fichier
      doc.end(); // Terminez le document
    })
    .catch((error) => {
      console.error('Erreur lors de la correction de l\'orientation de l\'image :', error);
    });
}

Avec cette modification, nous utilisons la bibliothèque sharp pour corriger l'orientation de l'image avant de l'ajouter au PDF, ce qui devrait résoudre le problème de rotation incorrecte.

Si vous ne souhaitez pas utiliser la bibliothèque sharp pour corriger l'orientation de l'image, vous pouvez essayer d'utiliser la bibliothèque exif pour lire les données EXIF de l'image et déterminer son orientation. Ensuite, vous pouvez ajuster les coordonnées de l'image en conséquence. Voici comment vous pouvez le faire :

javascript
const PDFDocument = require('pdfkit');
const fs = require('fs');
const exif = require('exif').ExifImage; // Importez la bibliothèque exif

// Créez un nouveau document PDF
const doc = new PDFDocument();
const picturePath = 'chemin/vers/votre/image.jpg';

// Définissez les coordonnées du cercle
const circleX = 125;
const circleY = 130;
const circleRadius = 60;

// Définissez le padding pour l'image
const padding = 5;

// Définissez les coordonnées de l'image (en ajoutant le padding)
let imageX = circleX - circleRadius + padding;
let imageY = circleY - circleRadius + padding;
const imageWidth = (circleRadius - padding) * 2;
const imageHeight = (circleRadius - padding) * 2;

// Définissez la couleur de la bordure du cercle
const whiteStrokeColor = "#FFFFFF";
doc.strokeColor(whiteStrokeColor);

// Dessinez le cercle
doc
  .circle(circleX, circleY, circleRadius)
  .lineWidth(2) // Largeur de la bordure
  .stroke();

// Chargez l'image à l'intérieur du cercle
if (fs.existsSync(picturePath)) {
  // Lisez les données EXIF de l'image pour obtenir l'orientation
  try {
    new exif({ image: picturePath }, function (error, exifData) {
      if (!error && exifData) {
        const orientation = exifData.image.Orientation || 1;

        // Ajustez les coordonnées de l'image en fonction de l'orientation
        if (orientation === 6) {
          // Rotation de 90 degrés (vers la droite)
          imageX += imageHeight;
        } else if (orientation === 8) {
          // Rotation de 90 degrés (vers la gauche)
          imageY += imageWidth;
        } else if (orientation === 3) {
          // Rotation de 180 degrés
          imageX += imageWidth;
          imageY += imageHeight;
        }

        // Appliquez un masque circulaire à l'image
        doc.save()
          .circle(circleX, circleY, circleRadius - padding) // Créez un masque circulaire avec le padding
          .clip() // Appliquez le masque
          .image(picturePath, imageX, imageY, { width: imageWidth, height: imageHeight }) // Ajoutez l'image
          .restore(); // Restaurez l'état du document pour supprimer le masque

        // Enregistrez ou affichez le document PDF comme nécessaire
        doc.pipe(fs.createWriteStream('output.pdf')); // Enregistrez le PDF dans un fichier
        doc.end(); // Terminez le document
      } else {
        console.error('Erreur lors de la lecture des données EXIF :', error);
      }
    });
  } catch (error) {
    console.error('Erreur lors de la lecture des données EXIF :', error);
  }
}

Avec cette approche, nous utilisons la bibliothèque exif pour lire les données EXIF de l'image et déterminer son orientation. Ensuite, nous ajustons les coordonnées de l'image en fonction de l'orientation détectée. Cela devrait résoudre le problème de rotation incorrecte sans utiliser sharp. Assurez-vous d'installer la bibliothèque exif en exécutant npm install exif si vous ne l'avez pas déjà fait.

const PDFDocument = require('pdfkit');
const fs = require('fs');
const Jimp = require('jimp'); // Importez la bibliothèque jimp

// Créez un nouveau document PDF
const doc = new PDFDocument();
const picturePath = 'chemin/vers/votre/image.jpg';

// Définissez les coordonnées du cercle
const circleX = 125;
const circleY = 130;
const circleRadius = 60;

// Définissez le padding pour l'image
const padding = 5;

// Définissez les coordonnées de l'image (en ajoutant le padding)
let imageX = circleX - circleRadius + padding;
let imageY = circleY - circleRadius + padding;
const imageWidth = (circleRadius - padding) * 2;
const imageHeight = (circleRadius - padding) * 2;

// Définissez la couleur de la bordure du cercle
const whiteStrokeColor = "#FFFFFF";
doc.strokeColor(whiteStrokeColor);

// Dessinez le cercle
doc
  .circle(circleX, circleY, circleRadius)
  .lineWidth(2) // Largeur de la bordure
  .stroke();

// Chargez l'image à l'intérieur du cercle après correction de l'orientation
if (fs.existsSync(picturePath)) {
  // Utilisez Jimp pour corriger l'orientation de l'image
  Jimp.read(picturePath)
    .then((image) => {
      // Ajustez les coordonnées de l'image en fonction de l'orientation
      if (image.exif && image.exif.Orientation) {
        switch (image.exif.Orientation) {
          case 6:
            // Rotation de 90 degrés (vers la droite)
            image.rotate(90);
            break;
          case 8:
            // Rotation de 90 degrés (vers la gauche)
            image.rotate(-90);
            break;
          case 3:
            // Rotation de 180 degrés
            image.rotate(180);
            break;
        }
      }

      // Convertissez l'image Jimp en tampon (buffer)
      return image.getBufferAsync(Jimp.MIME_JPEG);
    })
    .then((buffer) => {
      // Appliquez un masque circulaire à l'image
      doc.save()
        .circle(circleX, circleY, circleRadius - padding) // Créez un masque circulaire avec le padding
        .clip() // Appliquez le masque
        .image(buffer, imageX, imageY, { width: imageWidth, height: imageHeight }) // Ajoutez l'image
        .restore(); // Restaurez l'état du document pour supprimer le masque

      // Enregistrez ou affichez le document PDF comme nécessaire
      doc.pipe(fs.createWriteStream('output.pdf')); // Enregistrez le PDF dans un fichier
      doc.end(); // Terminez le document
    })
    .catch((error) => {
      console.error('Erreur lors de la manipulation de l\'image :', error);
    });
}

Cette solution utilise le module jimp pour la correction d'orientation de l'image, tout en évitant l'utilisation de bibliothèques externes spécifiques à la manipulation d'images comme sharp. Assurez-vous d'installer jimp en exécutant npm install jimp si vous ne l'avez pas déjà fait.
